# Chat Box

## Các bước để chạy dự án

#### B1: Lấy code từ git

#### B2: cd vào dự án

```bash
cd ChatBox
```

#### B3: cd vào Backend tải thư viện và chạy server

```bash
cd Backend
```

```bash
npm install
```

```bash
npm run dev
```

#### B4: cd .. ra thư mục ChatBox

```bash
cd ..
```

#### B5: cd vào Frontend tải thư viện và chạy dự án

```bash
cd Frontend
```

```bash
npm install
```

```bash
npm run dev
```
